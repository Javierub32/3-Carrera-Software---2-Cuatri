1.  docker build -t env .

2.  docker run --rm env  -> Faltan variables de entorno

3.  docker run --rm -e DB_HOST="1" -e DB_PORT="2" -e DB_USER="3" -e DB_PASSWORD="4" env  -> Imprime correctamente las variables

--------------------------------

Comandos:

-e nombre=valor ---> Establece una variable de entorno dentro del contenedor
