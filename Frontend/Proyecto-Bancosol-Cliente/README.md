# Proyecto Bancosol - Cliente

## Cómo ejecutar el proyecto

1. Clona el repositorio en tu máquina.
2. Ejecuta `npx http-server` en la raíz del proyecto para servir los archivos HTML.
3. Abre tu navegador y navega a `http://localhost:8080/html/campanyas.html` para ver la página principal de gestión de campañas de recogida de alimentos.
